## Установка и запуск
1. Клонирование репозитория.
2. Установка зависимостей:
```bash
npm install
```
3. Подготовка ресурсов:
Нужны logo.png и icon.png в src/ для иконки и логотипа.
4. Создание базы данных и импорт данных:
```bash
node init_db.js
node import_data.js
```
5. Запуск приложения
```bash
npm start
```
